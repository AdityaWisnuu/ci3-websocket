âš ï¸ TROUBLESHOOTING:

Problem: Port already in use
Solution: Change port in application\config\websocket.php

Problem: Can't connect from browser
Solution: 
  1. Verify server is running (Step 2)
  2. Check Windows Firewall
  3. Try connecting to ws://127.0.0.1:8080

Problem: PHP socket errors
Solution: Verify PHP sockets extension is enabled
  Run: php -m | findstr sockets
